# 服务端部署

## 注意事项

- ⚠️ **尽量避免和其他业务共用服务器**，避免冲突
- ⚠️ **不要更改安装脚本与文件**，会导致部署失败

## 安装步骤

1. 下载安装包并解压：

```bash
unzip cove-v26.0410.1911125-linux-amd64.zip
```

2. 进入解压目录，执行安装脚本：

```bash
cd cove-v26.0410.1911125-linux-amd64/
./setup.sh
```

3. 按提示操作（一路回车即可使用默认配置）

4. 安装完成后访问管理后台，使用 `Setup Token` 初始化管理员密码

## 快速配置

### 登录管理后台

访问 `http://<服务器IP>:52001`，使用安装时生成的 Setup Token 设置管理员密码。

> 默认用户名为 `admin`

### 导入 License

管理后台 → 企业设置 → 授权管理

将服务器设备指纹（ID）发给我们获取 License 文件，上传即可。

![导入License](./images/部署-导入License.png)

### 配置大模型

管理后台 → 模型配置

系统内置了多个模型平台，可以直接配置使用。

![模型供应商列表](./images/部署-模型供应商.png)

### 客户端发布

管理后台 → 版本升级

- 检查客户端接口请求地址是否正确
- 上传插件安装包后，用户重启 WPS 即可自动静默升级
- 支持在线安装和命令行安装两种方式

![版本升级界面](./images/发布-版本升级界面.png)

### 品牌定制

管理后台 → 企业设置 → 品牌定制

可自定义产品标识、品牌色调与企业信息，修改后客户端界面同步更新。

![品牌定制](./images/发布-品牌定制.png)
