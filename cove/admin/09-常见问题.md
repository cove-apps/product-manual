# 常见问题

## 部署问题

**Q：需要什么样的服务器？**

建议 8 核 CPU、16GB 内存、256GB 硬盘的 Linux 服务器。尽量避免和其他业务共用。

**Q：支持哪些操作系统？**

服务端支持 Ubuntu 20.04+、CentOS 7+、Redhat 7+，以及麒麟 V10、统信 V20 等国产系统。

**Q：安装脚本执行失败怎么办？**

- 确认系统版本在适配列表中
- 不要修改安装脚本的原始内容
- 如果提示权限问题，使用 `sudo ./setup.sh`

## 模型配置

**Q：如何选择使用哪个模型？**

- 文档处理推荐 Qwen3.5-27B，不建议用推理模型（等待时间长）
- 识图推荐 DeepSeek OCR 或 Qwen2.5-VL

**Q：模型测试连接失败**

- 确认 API Key 是否正确
- 确认网络连通性（内网模型检查地址和端口）
- 检查模型是否已加载完成

## 用户管理

**Q：用户无法登录**

- 检查用户是否已通过审批
- 检查用户账号是否被禁用
- 确认服务器地址配置正确

**Q：如何让用户注册后直接可用？**

在管理后台 → 用户管理 → 自动审批规则中开启自动审批。

## 运维问题

**Q：如何升级服务端？**

管理后台 → 版本升级 → 上传新版本 → 系统自动部署。

**Q：日志文件太大怎么办？**

在操作日志页面选择备份或清理，按时间段清理历史日志。

> ⚠️ 清理的日志无法恢复。

**Q：License 过期了怎么办？**

在管理后台 → 企业设置 → 授权管理中查看设备指纹，联系获取新 License。
