# 技能库

## 什么是 Skill（技能）

Skill 是 AI 智能体中用于执行特定任务的标准化、可复用的能力模块。

> 简单理解：**快捷指令是"一句话"让 AI 做事，Skill 是"一整套流程"让 AI 做事**。

对于非标准的、复杂的排版、校审或其他定制化需求，通常需要用 Skill 来实现。

## 内置 Skill

系统内置了以下 Skill，可直接使用：

| Skill | 用途 |
|------|------|
| **GBT9704** | GB/T 9704-2012 国标公文排版 |
| **GBT7713-2** | GB/T 7713.2-2022 学术论文排版 |
| **template-creator** | 模板生成器 —— 从文档提取格式 |
| **quick-typeset** | 极速排版 —— 套用已有模板 |
| **doc-cleaner** | 文档清理 |
| **proofread-national-standards** | 国家标准引用校对 |
| **proofread-with-profile** | 自定义标准校对 |

## Skill 管理

管理员可在后台上传、编辑、删除 Skill，并设置可用范围（按用户角色配置）。

> 复杂技能的编排和制作可联系 OfficeAI FDE 服务顾问。

## 用户自建 Skill

客户端用户也可以在 Skill 编辑器页面中自行创建 Skill，支持：
- WYSIWYG（所见即所得）编辑
- 打包导入/导出
- 本地安装和管理
