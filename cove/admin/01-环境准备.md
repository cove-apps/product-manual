# 环境准备

企业版安装需要在本地服务器上部署服务端，再将客户端下载链接发送给用户安装使用。

## 硬件要求

| 配置项 | 最低要求 |
|--------|---------|
| CPU | 8 核 |
| 内存 | 16 GB |
| 硬盘 | 256 GB |

## 软件适配

### 客户端

| 平台 | 支持版本 |
|------|---------|
| Windows | Windows 7（SP1, 64位）及以上 |
| 信创系统 | 麒麟 Kylin V10、统信 UOS V20 及以上 |
| macOS | X86、M 系列芯片架构 |
| 办公软件 | Microsoft Office 2013、WPS 2019 及以上 |

### 服务端

| 平台 | 支持版本 |
|------|---------|
| Linux | Ubuntu 20.04、CentOS 7、Redhat 7 及以上 |
| 信创系统 | 麒麟 Kylin Server V10、统信 UOS Server V20 及以上 |

## 大模型

| 部署方式 | 说明 |
|---------|------|
| **内网模型** | 本地部署的 Qwen、DeepSeek 等开源模型，可通过 vLLM、Ollama、LMStudio 等框架搭建 |
| **外网模型** | 阿里云、腾讯云、华为云、火山引擎、硅基流动、月之暗面等平台提供的云端服务 |
| **专网模型** | 政务云、公安网、海关网等行业专网提供的大语言模型服务 |

## 模型推荐

| 场景 | 推荐模型 | 说明 |
|------|---------|------|
| 通用文档处理 | **Qwen3.5-27B** | 不建议用 DeepSeek R1 等推理模型，思考过程等待时间长 |
| 识图场景 | **DeepSeek OCR** / **Qwen2.5-VL** | DeepSeek OCR 更轻更快，QwenVL 多模态能力更强 |
