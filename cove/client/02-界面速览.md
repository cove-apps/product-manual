function playCoveVideo(el) {
  var container = el.parentNode;
  var url = el.getAttribute('data-video-url');
  if (!url) return;
  el.innerHTML = '<div style="padding:20px;color:#666;">加载中...</div>';
  el.style.cursor = 'default';
  el.onclick = null;
  var xhr = new XMLHttpRequest();
  xhr.open('GET', url, true);
  xhr.responseType = 'blob';
  xhr.onload = function() {
    if (xhr.status === 200) {
      var video = document.createElement('video');
      video.controls = true;
      video.style.width = '100%';
      video.style.maxWidth = '720px';
      video.style.borderRadius = '8px';
      video.style.display = 'block';
      video.src = URL.createObjectURL(xhr.response);
      container.innerHTML = '';
      container.appendChild(video);
      video.play().catch(function(){});
    } else {
      el.innerHTML = '<div style="color:red;padding:20px;">加载失败，请重试</div>';
    }
  };
  xhr.onerror = function() {
    el.innerHTML = '<div style="color:red;padding:20px;">网络错误，请重试</div>';
  };
  xhr.send();
}
</script>

## 任务类型按钮

![任务类型按钮](./images/界面-任务类型按钮.png)

- 清晰呈现核心功能入口，如「校对」「排版」「翻译」等
- 点击某个任务类型，下方会显示该任务下的快捷指令
- **管理员可在后台自定义**按钮的顺序、图标、名称和功能

## 快捷指令按钮

![快捷指令按钮](./images/界面-快捷指令按钮.png)

- 点击任务类型后，对应的快捷指令会显示在界面下方
- 快捷指令是某个任务下的具体功能。例如「校对」任务下，有「常规纠错」「公文写作规范」等
- 你可以在界面右上角点击「+」添加个人快捷指令，保存常用提示词

## 自然对话窗口

![自然对话窗口](./images/界面-自然对话窗口.png)

- 在底部的输入框中输入文字，与 AI 进行自然对话
- 输入 `/` 可快速选择所有快捷指令
- 适合对模型生成结果反复沟通，直到满意为止

## 模型选择

![模型选择按钮](./images/界面-模型选择按钮.png)

- 如果管理员接入了多个 AI 模型，你可以在此切换
- 不同模型擅长不同的任务：参数大的模型思考能力强，但响应也会慢一些

## 更多操作

![更多操作按钮](./images/界面-更多操作按钮.png)

- **添加附件**：上传图片或文稿让 AI 识别或处理
- **关联文档**：将多个文档关联，便于对比或合并
- **思考过程**：关闭后可让简单任务的处理速度更快

## 开启新对话

![开启新对话按钮](./images/界面-开启新对话.png)

- 点击右上角「开启新对话」按钮
- 可以从任何页面返回插件首页，重新选择任务类型或进行自由对话
