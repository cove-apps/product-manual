# 校对

> 用 AI 找出文档中的错别字、语法错误等并进行纠正。

人工校对费时费力还容易遗漏——一篇文章看三遍还是漏了错别字。AI 校对几秒扫完一篇文档，不放过任何一个细节。行政、法务、研发、设计，凡是需要输出正式文档的人，都用得上。

## 使用方式

1. 打开一篇需要校对的文档
2. 点击任务类型中的「校对」
3. 选择一条快捷指令，或直接在输入框输入校对需求
4. 等待处理完成，弹出确认窗口，逐处审阅修改

## 内置快捷指令

| 指令 | 说明 |
|------|------|
| **常规纠错** | 纠正错别字、大小写、标点符号错误 |
| **公文写作规范** | 按公文写作规范校对数字、标点、文种格式 |
| **设计文档编号校对** | 检查标题、图、表、公式编号的连续性和格式 |
| **领导人职务及排序** | 检查领导人姓名、职务、排序规范（虚拟演示数据） |

> 以上指令为初始配置，管理员可在后台增删调整。你也可以点「+」添加个人快捷指令。

## 国家标准引用校对

安装对应技能（Skill）后，可识别文档中国家标准引用的 5 类错误：编号格式错误、名称与编号不匹配、有名无号、有号无名、标准过期或已被替代。适合工程、设计、公文类文档。

## 自定义校对方案

企业可配置专属校对方案，按内部规范进行批量校对——例如设计院的水电站规范校对。联系管理员配置。
