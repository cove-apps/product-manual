# 小工具

> 处理各种小而繁琐的文档操作，一键完成。

## 删除多余空行

1. 打开需要清理的文档
2. 点击「小工具」→「删除多余空行」
3. AI 自动识别并删除多余空行

![删除空行](./images/小工具-删除空行.png) ![已删除](./images/小工具-空行已删除.png)

## 表格统一对齐

1. 打开包含表格的文档
2. 点击「小工具」→「表格统一」
3. 自动调整文档中所有表格的对齐方式

![表格对齐](./images/小工具-表格对齐.png) ![已对齐](./images/小工具-表格已对齐.png)

## 其他小工具

| 功能 | 说明 |
|------|------|
| **一键清除格式** | 清除文档中所有多余格式 |
| **一键删除批注** | 删除文档中的所有批注 |
| **文档清理** | 按规则批量删除英文段落、URL、邮箱、空行等 |

> 小工具可按你的日常需求自定义增加，尝试用自然语言描述你的需求即可。
