# 小工具

> 各种小而繁琐的文档操作，一键完成。

文档操作中总有那些反复出现的小麻烦——空行太多要逐段删、表格对不齐要手动调、页眉横线去不掉、多份文档合并复制粘贴到崩溃。小工具把这些琐碎操作打包，点一下就能解决。人人都会用到，是所有功能中使用频率最高的。

## 使用方式

1. 打开需要操作的文档
2. 点击任务类型中的「小工具」
3. 选择需要的工具指令（如删除空行、表格统一等）
4. AI 一键完成操作

## 内置指令

| 指令 | 说明 |
|------|------|
| **多文档合并** | 将多个已打开的文档合并到一篇，保留原有样式 |
| **多文档对比** | 对比两个文档，在激活文档中用批注标出差异 |
| **一键清除所有格式** | 清除全文所有样式，恢复纯文本 |
| **删除文中所有批注** | 一键删除文档中所有批注 |
| **表格统一** | 全文所有表格自适应内容与窗格 |
| **删除多余空行** | 删除文档中的所有空白段落 |
| **删除页眉横线** | 删除所有页眉中的横线 |

## 使用场景示例

**合并文档：**
1. 在 WPS 中打开所有需要合并的文档
2. 点击「小工具」→「多文档合并」
3. 选择目标文档，AI 自动合并

**对比文档：**
1. 打开两份需要对比的文档
2. 点击「小工具」→「多文档对比」
3. AI 在当前文档上用批注标出差异点
