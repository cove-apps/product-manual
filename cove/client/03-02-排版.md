# 排版

> 一键将文档排版为符合要求的格式，包括字体、行间距、编号、页边距等。

格式混乱的文档读起来不专业，手动调整又琐碎耗时。排版功能让文档格式一键统一——无论公文红头、会议纪要还是学术论文，都能自动套用标准模板，排版结果稳定可复现。对行政、设计、研发团队来说，告别逐行调格式的重复劳动。

![排版操作](./images/排版-操作前.png)

## 使用方式

1. 打开需要排版的文档（或选中要排版的区域）
2. 点击任务类型中的「排版」
3. 选择模板排版一键套用，或精细排版按需调整单项格式
4. 等待处理完成

## 模板排版

选择模板后一键套用，适合公文、会议纪要、学术论文等固定格式文档。

| 指令 | 说明 |
|------|------|
| **企业通用公文** | 企业日常公文（通知、报告、请示等），自动识别标题、正文、落款 |
| **企业会议纪要** | 会议纪要红头格式，添加红头横线和版记横线 |
| **党政公文格式** | GB/T 9704-2012 国标公文排版，支持通用/信函/命令/纪要四种格式 |
| **设计文档一键排版** | 标题黑体、正文宋体，图名/表名/页眉页脚全部规范 |

## 精细排版

按需调整文档的特定格式，不改变其他内容。

| 功能 | 说明 |
|------|------|
| **排版图** | 图名黑体小五居中，图片居中对齐 |
| **排版表格** | 表格与页面等宽，表头加粗，跨页重复表头 |
| **清除格式** | 清除全文或选区的所有样式，恢复纯文本 |
| **页面设置** | 调整纸张大小、方向、页边距、首页不同、奇偶页不同 |
| **刷新域/目录** | 更新页码、日期、交叉引用、目录等域代码 |

## 极速排版

排版引擎可在 1 分钟内完成 50 万字级长文档排版，适合已保存的 .docx 长文档。

## 模板管理

- **提取模板**：从一份格式规范的文档中提取排版模板
- **套用模板**：将已有模板一键应用到当前文档
- 模板资产由管理员在后台统一管理分发
