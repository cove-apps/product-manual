# 翻译

> 支持多语种翻译，译文与原文对照显示，方便逐条确认。

涉外业务越来越多——找翻译公司贵、自己翻又不确定专不专业。AI 翻译支持 6 种语言互译，译文与原文逐段对照，保留文档原有格式。适合跨国项目文档、外语合同、技术手册等场景，省去外包沟通成本。

## 使用方式

1. 打开一篇需要翻译的文档
2. 点击任务类型中的「翻译」
3. 选择目标语言（如「翻译为英语」）
4. 等待处理完成，文档中会生成对照译文

## 内置语种

| 语种 | 说明 |
|------|------|
| **英语** | 中英互译 |
| **中文** | 英中互译 |
| **日语** | 中日互译 |
| **韩语** | 中韩互译 |
| **法语** | 中法互译 |
| **阿拉伯语** | 中阿互译，支持 RTL 从右到左排版 |

## 新增语种

如果需要的语种不在内置列表中：

1. 点击快捷指令旁的「+」
2. 输入自然语言描述，如「将文档翻译成葡萄牙语」
3. 保存为快捷指令，方便后续调用

## 技术说明

- 支持选区翻译和全文翻译两种模式
- 任务中途取消可整篇恢复到原文状态
