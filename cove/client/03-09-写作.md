# 写作

> 根据你的需求生成各类文稿，支持公文、协议、方案、报告等多种场景。

面对空白文档无从下手？写作功能提供 10 种内置场景模板，从党建会议纪要到项目投标文件，AI 帮你搭框架、填内容。不满意可以反复沟通调整，结果满意了再导入文档。适合需要高频产出正式文稿的岗位——行政、法务、商务、招投标专员，告别憋稿焦虑。

## 使用方式

![写作指令面板](./images/写作-指令面板.png)

1. 点击任务类型中的「写作」
2. 选择一条快捷指令，或直接在输入框描述你的写作需求
3. AI 生成文稿内容，展示在对话中
4. 可多次沟通调整，满意后导入文档

## 内置指令

| 指令 | 适用场景 |
|------|---------|
| **党建会议纪要** | 党委会纪要，决议事项格式化呈现 |
| **技术保密协议** | 涉密人员行为清单、数据跨境限制、泄密追责条款 |
| **廉洁活动方案** | 廉洁文化月实施方案，含警示教育、家风建设、廉政测试 |
| **总结报告** | 专项工作总结，突出量化指标和成果 |
| **项目投标文件** | 投标方案标书，含成功案例和成本测算 |
| **项目申报书** | 各类项目申报材料 |
| **技术报告** | 技术研究或分析报告 |
| **软件采购申请** | 采购申请文书，含价值论证和量化指标 |
| **招标文件** | 设备采购招标文件 |
| **采购招标文件** | 采购招标公告，含技术标准和惩罚条款 |

## 自定义写作

- 如果内置指令不完全匹配你的场景，直接在输入框描述需求即可
- 也可以将常用的写作需求保存为个人快捷指令，方便重复调用
