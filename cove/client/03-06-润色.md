# 润色

> 将文档行文按不同风格进行润色处理：更正式、更活泼、更学术，甚至自定义风格。

内容写完了但语气不对味？润色帮你精准调整文档风格——正式场合庄重得体、党政公文规范严谨、学术论文专业可靠、新媒体传播活泼吸睛。适合需要面向不同读者调整表达的岗位：行政、品牌、市场、科研，人人都用得上。

## 使用方式

1. 打开需要润色的文档
2. 点击任务类型中的「润色」
3. 选择一种润色风格
4. 等待处理完成，预览润色效果

## 内置风格

| 风格 | 说明 |
|------|------|
| **正式风格** | 官方书面语言，结构清晰，逻辑紧密 |
| **党政风格** | 政府公文风格 |
| **学术风格** | 严谨、科学的专业报告模式 |
| **活泼风格** | 活泼可爱，融入幽默元素 |
| **口语化风格** | 通俗易懂，说人话 |
| **鲁迅小说体** | 仿鲁迅文风，简练深刻 |

## 自定义风格

- 点击「+」新增自己的润色风格
- 输入自然语言描述，如「用王家卫电影旁白的风格润色」
- 保存后下次可直接调用

> 管理员也可以在后台配置企业专属润色风格。
